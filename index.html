Descrição Informal
Resumo
O projeto visa desenvolver um aplicativo móvel para o registro e acompanhamento de treinos de musculação, direcionado especificamente aos usuários de academias. Este aplicativo oferecerá uma experiência abrangente, permitindo não apenas o registro dos treinos, mas também a compreensão aprofundada dos dados gerados durante as sessões de musculação.
Motivação
Ao iniciar na academia, muitas pessoas se sentem perdidas, especialmente com a diminuição de professores disponíveis para orientação. Além disso, as ferramentas existentes para acompanhar a evolução no treino são, na maioria das vezes, pouco intuitivas e ineficientes. Muitas fornecem informações excessivas, tornando a utilização durante o treino desafiadora, enquanto outras são excessivamente simples, oferecendo apenas funcionalidades básicas de um bloco de notas. Nosso aplicativo visa preencher essa lacuna, proporcionando uma solução intuitiva e eficaz para o registro e acompanhamento de treinos, facilitando a jornada de evolução dos usuários na academia.
Sistema

a. Propomos um sistema inovador que disponibiliza uma lista de exercícios pré-cadastrados para que os usuários possam personalizar seus planos de treino. Eles terão a flexibilidade de criar e gerenciar vários planos de treino, todos vinculados à sua conta individual.

 

b. Ao acessar a conta, o usuário será recebido por uma lista de treinos. Ao selecionar um treino específico, a interface apresentará uma seção pronta para o treino, exibindo automaticamente o primeiro exercício. Nesta tela, será fornecido o nome do exercício, uma animação representativa, um diagrama destacando os músculos envolvidos, a carga utilizada no último treino, o número de repetições e séries programadas. Botões intuitivos permitirão ao usuário finalizar uma série (solicitando imediatamente a carga usada para atingir a falha), concluir o exercício e, ao término, avaliar o nível de fadiga numa escala de 1 a 5.

 

c. A seção de análise consolidará todos os dados coletados em cada sessão de treino, gerando relatórios visuais da progressão ao longo do tempo. Gráficos informativos destacarão áreas de destaque e possíveis melhorias. Além disso, com base nessas informações, o sistema fornecerá sugestões personalizadas para otimizar ainda mais o programa de treinamento do usuário.
